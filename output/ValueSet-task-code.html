ValueSet: OH eForms Task Code ValueSet

Official URL: http://ontario.ca/fhir/eforms/ValueSet/task-code Version: 0.1.0
Active as of 2025-07-07 Computable Name: EFormsTaskCodeVS

Allowed codes for Task.code in the eForms implementation guide. Combines base FHIR Task codes, SDC codes, and eForms-specific codes.
